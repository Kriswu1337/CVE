# EDIMAX RCE

**description**: a remote-command-execution vulnerability was found on EDIMAX BR-6478AC V3 via the sub_44CCE4 function and syscmd in /bin/boa

## Firmware

**brand**:edimax  

**product**:AC1200 Wave 2 Dual-Band Gigabit Router  

**version**:BR-6478AC V3 1.0.15

The firmware can be downloaded from this website[1](https://www.edimax.com/edimax/merchandise/merchandise_detail/data/edimax/global/home_legacy_wireless_routers/br-6478ac_v3) and using FirmAE to simulate the router environment.  The command is

```sudo ./run.sh -d edimax ../Desktop/FIRWARE/BR-6478AC_v3_1.0.15.bin ```
